# PWA Icons

Add the following icon files for PWA support:

- `icon-192x192.png` - 192x192 pixel icon
- `icon-512x512.png` - 512x512 pixel icon
- `apple-touch-icon.png` - 180x180 pixel icon for iOS

These will be implemented in T9 (PWA polish).

For now, you can use placeholder images or simple colored squares.
